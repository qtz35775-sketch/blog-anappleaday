Hexo 博客搭建指南
2026年4月28日

最近花了一些时间搭建了这个基于 Hexo 的个人博客，在这里记录一下搭建过程和心得体会。

为什么选择 Hexo

Hexo 是一个快速、简洁且高效的博客框架。它使用 Markdown 来撰写文章，通过模板引擎生成静态页面，部署简单方便。

相比其他博客平台，Hexo 有以下优势：

  1. 速度快 - 基于 Node.js，生成速度非常快
  2. 易于部署 - 可以直接部署到 GitHub Pages
  3. 主题丰富 - 社区提供了大量精美的主题
  4. Markdown 支持 - 使用 Markdown 写作，专注于内容
