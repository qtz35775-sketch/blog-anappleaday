基于 Hexo 框架搭建的个人博客，使用 Frame 主题，支持文章分类、标签云、图片画廊等功能。

## 技术栈

- **框架**: Hexo
- **主题**: Frame（自定义修改）
- **部署**: GitHub Pages
- **域名**: anappleaday.top

## 功能特点

- 响应式布局，适配移动端
- 支持文章分类与标签管理
- 图片画廊展示
- 简洁优雅的阅读体验
