Markdown 是一种轻量级标记语言，让你可以用纯文本格式来编写文档。今天来分享一些实用的 Markdown 技巧。

## 基础语法回顾

### 标题

使用 `#` 号来创建标题，一个 `#` 是一级标题，两个 `##` 是二级标题，以此类推。

### 文本样式

- **粗体** 使用 `**文本**`
- *斜体* 使用 `*文本*`
- ~~删除线~~ 使用 `~~文本~~`

### 列表

无序列表使用 `-` 或 `*`，有序列表使用数字加 `.`。

## 进阶技巧

### 代码块

使用三个反引号可以创建代码块，并指定语言实现语法高亮：

```javascript
function hello() {
  console.log("Hello, World!");
}
```

### 表格

| 功能 | 语法 | 效果 |
|------|------|------|
| 粗体 | `**文本**` | **文本** |
| 链接 | `[文本](URL)` | [文本](#) |

### 引用

## 总结

掌握 Markdown 的基础语法后，你就能高效地撰写博客文章了。关键是多练习，让语法成为你的肌肉记忆。
